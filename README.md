# Cadastro de séries (projeto de estudo)
Projeto de estudo feito em .NET para as aulas da [Digital Innovation One](https://dio.me)

Opções irão aparecer no console:  
  Informe a opção desejada:  
  1- Listar séries  
  2- Inserir nova série  
  3- Atualizar série  
  4- Excluir série  
  5- Visualizar série  
  C- Limpar Tela  
  X- Sair
